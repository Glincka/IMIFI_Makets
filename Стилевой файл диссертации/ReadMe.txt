Данный шаблон собран для написания бакалаврской/магистрской работы под нужды Института математики и фундаментальной информатики Сибирского федерального университета. При использовании шаблона предполагается, что у пользователя имеются некоторые навыки работы в редакторской системе LaTeX, поэтому данное руководство не затрагивает эти аспекты. Шаблон выполнен согласно стандарту Сибирского федерального университета СТУ 7.5-07-2021.

Основной файл, с которого начинается сборка проекта, является "main.tex". В файле "main.pdf" представлены основные команды для работы со стилевым файлом.
